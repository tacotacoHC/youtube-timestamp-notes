# YouTube Timestamp Notes (Chrome Extension)

## Install (Developer Mode)
1. Go to `chrome://extensions`
2. Turn on **Developer mode**
3. Click **Load unpacked**
4. Select this folder

## Usage
- Open a YouTube video (`/watch?v=...`)
- Click the **+ Note** floating button, write a note, save
- View/search notes via the extension popup
- Shortcut: **Alt+N** (Windows/Linux), **Option+N** (Mac)

## Data
Notes are stored in `chrome.storage.local` under key `ytTimestampNotes_v1`.
